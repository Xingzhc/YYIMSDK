用友有信 `WEB_SDK` 是一款解决 `web` 即时通讯的工具包，它包含基本功能包 YYIMSDK 和各功能模块扩展包（包含：摘要／下载／扩展／群组／消息／个人配置／公共号／联系人／待办／上传） 。
